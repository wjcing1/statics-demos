1. External Supports — how to count r

ED = r − 3
r = total number of reaction components from all supports (in 2D, DOFs = 3)
Support Symbol r Reactions Allowed movement
Roller
(simple / carrello)		 1 V Horizontal displacement + rotation
Pinned
(double / cerniera)		 2 H, V Rotation only
Fixed
(clamped / incastro)		 3 H, V, M None (fully restrained)
Slider
(fixed with roller)		 2 V, M Horizontal displacement
Double pendulum
(pendolo doppio)		 1 Along bar Perpendicular displacement + rotation

Source: Class 3, slides 16–17

2. Internal Degree (ID)

Start from a tree-alike structure (continuous, no closed parts, no releases) → ID = 0

Feature Symbol Effect on ID Explanation
Closed branch
(closed loop)		 +3 Each closed loop adds 3 redundant internal forces (N, V, M)
Internal hinge
(releases M = 0)		 −(n − 1) n = number of bars meeting at the hinge. Releases moment.
Axial release
(releases N = 0)		 −(n − 1) n = number of bars at the release. Releases axial force.

Source: Class 5, slides 9–11

3. DSI = ED + ID

DSI = ED + ID = (r − 3) + ID
DSI Classification Meaning
< 0 Hypostatic Mechanism — structure can move. Draw the failure mechanism!
= 0 Isostatic Statically determined — solvable with equilibrium equations only.
> 0 Hyperstatic Statically indeterminate — has redundant constraints. (Not solvable in this course.)
CAREFUL — formula can lie! (Class 5, slides 8 + 12)

Even if DSI ≥ 0, the structure is still hypostatic if:
• All reactions are parallel (e.g. all rollers in the same direction)
• All reactions are concurrent in one point

In these cases → the structure is a mechanism regardless of what the formula says. Always check!

4. Worked example

Example from Class 5, slide 13 + 15
Strategy 1:
ED = r − 3 = (3 + 2) − 3 = +2 (externally hyperstatic)
Closed branches: 0 → starts at +0
Internal hinges: 1 hinge, 2 bars → −(2−1) = −1
ID = 0 − 1 = −1 (internally hypostatic)
DSI = +2 + (−1) = +1 → check: wait...
Strategy 2:
Bars: 1 rigid body → DOFs = 1 × 3 = 3...
Actually this structure has 2 bars (split by hinge)
DOFs = 2 × 3 = 6
DOCs = 3(fixed) + 2(pin) + 2(hinge) = 7...
